### Is the drug target real?

Most target-discovery projects assume the target is worth it and optimize downstream. I invert that: I try to **falsify the target first** — with reproducible, content-addressed pipelines that publish their nulls. If a target survives a genuine attempt to break it, it earns the medicinal chemistry.
